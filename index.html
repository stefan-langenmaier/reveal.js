What's Gentoo

Gentoo is a source-based, rolling-release meta-distribution! :)

Terms and tools

  • portage is the repository of packages or so called ebuilds, complete copy locally
  • ebuild is a recipe how to create a package from the source files and how to install
  • emerge is a tool that executes theses ebuilds and does depdency resolution, depdency resolution is really tough in gentoo
Necessary tools ebuilds for example can provide also Firefox(TM) because it is not compiled

Terms and tools

  • eread - don't want to follow the installation, read about it later
  • etc-update - don't want to update the configuration now, do it when you have time, Gentoo never overwrites configurations
  • genkernel - don't remeber how to create a initramfs? me neither. have you ever create a symlink wrong?
helping tools

Terms and tools

  • eix - extras speed for emerge
  • gentoolkit - more search tools
  • revdep-rebuild - find executables with wrong/old/missing libraries, this tools finds it and reinstall their dependencies
