What do the references say Eclipse help Eclipse is a platform that has been designed from the ground up for building integrated web and application development tooling. At the core of Eclipse is an architecture for dynamic discovery, loading, and running of plug-ins. The platform handles the logistics of finding and running the right code. The platform UI provides a standard user navigation model. Each plug-in can then focus on doing a small number of tasks well. What kinds of tasks? Defining, testing, animating, publishing, compiling, debugging, diagramming...the only limit is your imagination. Wikipedia Eclipse is an integrated development environment (IDE)
