What's Gentoo

Gentoo is a source-based, rolling-release meta-distribution! :)

Terms and tools

  • portage, repository of packages or so called ebuilds, complete copy locally
  • ebuild is a recipe how to create a package from the source files and how to install
  • emerge is a tool that executes theses ebuilds and does depdency resolution, depdency resolution is really tough in gentoo
  • eselect profiles and alternatives
  • eread
  • etc-update
  • genkernel
  • eix
  • gentoolkit
  • revdep-rebuild

Only one version

* no re-installs, just has to installed once * bleeding edge falls back into stable (later we get to that) * not really true, stable, unstable, masked, slots, but there is only one current Gentoo
